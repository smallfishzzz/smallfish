# 目录结构
```bash 
├── LICENSE         # license file 
├── README.md       # this file 
├── _build          # build directory
├── doc             # source directory 
├── environment.yml # environment file 
└── source          # source directory 
```
# 维护命令

编译文档：
sphinx-build doc _build
